# Heapofstudents1 
Start the program

Print “Hello!” to greet the user.

Handle the student’s birth date

Create a Date object.

Initialize it with the birthdate from the CSV string.

Print the date in a readable format (e.g., February 17, 2004).

Handle the student’s address

Create an Address object.

Initialize it with street, city, state, and zip from the CSV string.

Print the address (street first, then city/state/zip).

Create the Student object

Make a Student object.

Split the CSV string by commas to get each piece of data: first name, last name, address info, birthdate, graduation date, and credits.

Store the address and dates inside the student object.

Print the student information

Print the full name.

Print the address.

Print date of birth and graduation date with labels.

Print the number of credits completed.

Print a divider line.

Print the student’s name in “Last, First” format.

Clean up and end

Delete any objects created with new.

End the program.
